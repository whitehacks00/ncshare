# ncshare
NCShare allows you to transfer and receive files locally or remotely using netcat.

![alt text](https://github.com/whitehacks00/ncshare/blob/main/img/ncshare.jpg)
# Requirements
* netcat
* netcat-openbsd
* nmap
# Installation
* chmod 777 ncshare.sh
* bash ncshare.sh
